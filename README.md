# ⚠️ HIGH-PERFORMANCE ANDROID RANSOMWARE ⚠️



A security research tool developed in Python for Android platforms that systematically identifies storage directories, locates specific file types based on predefined patterns, and applies cryptographic transformations to those files.



---

# 📱 Core Functions:


- Directory traversal across Android storage systems

- Pattern-based file identification

- Cryptographic file transformation
